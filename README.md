To-Do List Web App
This is a simple To-Do List application built using HTML, CSS, and Vanilla JavaScript.
It allows users to:
Add tasks
Mark tasks as complete
Delete tasks
View tasks instantly without page reload

Technologies Used
HTML
CSS
JavaScript (Vanilla JS)

Features
Dynamic task creation
Complete/Incomplete toggle
Remove task
Clean and responsive UI
